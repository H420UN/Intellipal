Intellipal — Autonomous Research Agent
Overview

Intellipal is an autonomous AI agent that performs structured research and decision-making from high-level user goals. It decomposes tasks, gathers evidence using tools, evaluates information quality, and generates explainable recommendations.

Features

Multi-agent architecture (Planner, Executor, Evaluator)

Tool-augmented LLM reasoning

Structured, deterministic outputs

CLI-based execution

Explainable decisions with source grounding

Example
python main.py \
  --goal "Find the best cloud provider for a fintech startup requiring SOC2 compliance"

Output

Recommended option

Evidence summary

Evaluation rationale

Confidence score

Tech Stack

Python

OpenAI API

Pydantic

Modular agent architecture

This project demonstrates real-world AI system design, agent orchestration, and tool-augmented reasoning — skills required for AI Engineer and ML Engineer roles.
